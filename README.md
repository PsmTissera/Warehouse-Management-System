 
# Warehouse Management System

## Overview
Warehouse Management System is a **Java web application** developed as a **group project** for the  
**Development of Enterprise Applications (DEA)** module.

The system supports warehouse operations such as viewing orders, managing reorder lists, and handling staff-related workflows using **Servlets and JSP**.

---

## Technologies
- Java (Servlets, JSP)
- MVC Architecture
- Apache Tomcat
- MySQL (JDBC)
- HTML, CSS
- Git & GitHub
